Les Displays

Display: Block

Cette div est un élément en Block. Il occupe toute la largeur de l'écran.

Exemple de balises de type "block":

  • div
  • p
  • h1,h2,h3,etc..
  • li
  • main
  • ul

Display: Inline

Les éléments en inline ne force pas le retour à la ligne, il prend simplement la place dont il a besoin

Attention !! Les élements de type inline ne sont pas comme les types block, leur hauteur et largeur ne sont délimités QUE par leur contenu

Exemple de balise de type inline

  • Balise à (liens)
  • img
  • input
  • video
  • span
  • label

Display: Inline-block

Inline-block: C'est un type inline qui conserve les propriétés du type block, il peut donc être modifié en hauteur et largeur comme n'importe quel type block

Display: Flex

Flex: les éléments dans le conteneur s'alignent en ligne et se repartissent l'espace disponible

Display: Grid

Grid: Avec grid on peut définir une grille avec des lignes et des colonnes

Display: none

None: Le display none cache tout simplement l'éléments.
